# discord-magiceden-tracker

simple node.js project to track sell/buy/listings from magiceden, sending notifications to discord

## Scripts
- `npm run dev` — start local dev server
- `npm run build` — build for production
- `npm start` — run production server
- `npm run lint` — lint

## Tech
- Next.js (App Router)
- React
- Tailwind CSS (+ daisyUI)
- Inter font via `next/font`

## SEO
- Canonical + metadata set in `app/layout.jsx`
- `app/robots.txt` and `app/sitemap.js`
- Base URL configurable via `.env.local` (`NEXT_PUBLIC_SITE_URL`)
